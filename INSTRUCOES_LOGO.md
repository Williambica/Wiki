# Instruções para Adicionar o Logo

## Passo 1: Salvar a Imagem do Logo

Salve a imagem do logo que você forneceu com o nome:
```
logo-farmacias-associadas.png
```

## Passo 2: Colocar na Pasta Public

Coloque o arquivo na pasta `public/` do projeto:
```
public/logo-farmacias-associadas.png
```

## Estrutura de Pastas

```
seu-projeto/
├── public/
│   └── logo-farmacias-associadas.png  ← Coloque aqui
├── src/
│   └── ...
└── ...
```

## Verificação

Após adicionar o logo, ele aparecerá automaticamente em:
- Landing Page (página inicial)
- Header (cabeçalho do site)
- Componente Logo

## Fallback

Se a imagem não carregar, o sistema usará automaticamente um logo SVG como fallback.
